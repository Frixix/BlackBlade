# BlackBlade - Landing Page para Barbería

BlackBlade es una landing page moderna diseñada para barberías que buscan mejorar su presencia digital y facilitar el agendamiento de citas con sus clientes.

Este proyecto está construido con HTML y CSS, enfocado en ofrecer una experiencia clara, atractiva y orientada a la conversión.

---

## Características

- Diseño moderno y profesional
- Adaptable a dispositivos móviles (responsive)
- Navegación por secciones
- Integración con WhatsApp para agendamiento
- Ubicación integrada con Google Maps
- Sección de servicios con precios
- Galería de trabajos
- Formulario de contacto
- Estructura escalable

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts (Inter)
- Font Awesome

## Personalización

Para adaptar este proyecto a un cliente real:

- Modificar el nombre de la barbería
- Ajustar precios y servicios
- Reemplazar contenido de la galería
- Actualizar el número de WhatsApp
- Editar ubicación en Google Maps
- Configurar horarios y datos de contacto

---

## Uso

Este proyecto puede utilizarse como:

- Plantilla para barberías
- Proyecto de portafolio
- Base para servicios web a negocios locales
- Landing page de captación de clientes
